# hand_gesture_recognition_system
A computer vision system that detects and recognizes 20+ hand gestures in real-time using webcam. Built with Python, OpenCV, and MediaPipe. Recognizes: Numbers (0-5), Letters (A,B,C,L,O,V), Shapes (Heart, Star, Gun), Actions (Thumbs Up/Down, High Five, Peace, Rock On) . Features: Real-time tracking, Left/Right hand detection, Gesture smoothing.
